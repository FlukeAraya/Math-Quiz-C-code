🧮 Math Speed Game (C)
Math Speed Game เป็นโปรแกรมเกมฝึกคณิตศาสตร์ด้วยภาษา C
ผู้เล่นจะต้องตอบคำถามบวก/ลบให้เร็วและถูกต้อง ภายในจำนวนข้อที่กำหนด
มีระบบจับเวลาและจัดอันดับ (Ranking) ตามคะแนนและความเร็ว

🎮 Features
* 🎲 สุ่มโจทย์คณิตศาสตร์ (Addition / Subtraction)
* 📊 มี 3 ระดับความยาก

  * Easy (เลข 1 หลัก)
  * Medium (เลข 2 หลัก)
  * Hard (เลข 3 หลัก)
* ⏱ จับเวลาในการทำข้อสอบ
* 🏆 ระบบจัดอันดับคะแนน (A - E)
* ⚡ ระบบจัดอันดับความเร็ว (Time Rank)
* 🔍 ตรวจสอบ input ป้องกันค่าที่ไม่ใช่ตัวเลข

🛠 Technologies Used
* Language: C
* Libraries:
  * stdio.h
  * stdlib.h
  * time.h
  * string.h
  * conio.h

▶️ How to Run
1. Compile โปรแกรม:

gcc main.c -o game

2. Run:

./game

🧠 How It Works

1. ผู้ใช้เลือกโหมด (Easy / Medium / Hard)
2. เลือกจำนวนข้อ (5 - 100)
3. โปรแกรมสุ่มคำถาม (+ หรือ -)
4. รับคำตอบจากผู้ใช้
5. ตรวจคำตอบและเก็บคะแนน
6. เมื่อจบ:
   * แสดงคะแนน
   * แสดง Rank
   * แสดงเวลาที่ใช้

Question 1 : 5 + 3 = ?
ANSWER : 8
